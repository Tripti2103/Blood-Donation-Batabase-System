# Blood Donation Database Management System (SQL + Python)
A complete SQL-based Database Management System designed to manage blood donors, blood banks, hospitals, and blood inventory. This project demonstrates end-to-end database design, SQL query development, schema creation, normalization, and connectivity using Python.
## Project Overview
The Blood Donation Database Management System (BDDMS) is built to streamline and organize data related to blood donors, blood packets, hospitals, and blood banks.
The system ensures easy retrieval, management, and storage of all crucial information using a relational database approach.

This project was developed as part of the DBMS Mini Project and includes:
  - Schema Diagram
 - Normalized tables
 - SQL queries (DDL, DML, joins)
 - Python-based frontend for CRUD operations
 - MySQL backend with foreign keys & constraints

## Tech Stack
 Backend (Database)
  - MySQL
  - SQL (DDL, DML, Joins, Constraints)

 Frontend / Connectivity
  - Python
  - Tkinter (GUI)
  - MySQL Connector
